Security Features

RAII Resource Safety Socket and memory resources managed via destructors and smart pointers.
Input Validation Path traversal, null bytes, and oversized requests rejected at the parser.
Process Sandboxing Form handling in a forked process with seccomp-bpf syscall filtering.
Audit Logging Thread-safe logging at five severity levels for forensic analysis.

Submissions are processed in a sandboxed child process and saved to disk.
